# UML Editor (Java Swing)

This is a compact Java Swing baseline for the OOP UML Editor assignment.

## Included files

- Main.java
- EditorFrame.java
- EditorCanvas.java
- EditorController.java
- DiagramRenderer.java
- ToolbarPanel.java
- LabelDialog.java
- ToolType.java
- ToolManager.java
- EditorModel.java
- DiagramElement.java
- Node.java
- BasicNode.java
- RectNode.java
- OvalNode.java
- CompositeNode.java
- Port.java
- Link.java
- GeometryUtil.java

## Compile

```bash
javac *.java
```

## Run

```bash
java Main
```

## Current baseline features

- Select / hover
- Temporary Rect / Oval creation flow
- Link creation by ports
- Single selection and drag-selection rectangle
- Move selected node
- Resize basic nodes from ports
- Group / Ungroup from Edit menu
- Label dialog from Edit menu

## Notes

This version is designed as a structured baseline. It compiles and provides the main interaction skeleton, but you may still want to refine behavior and visuals to match your course PDF exactly.
